Darwin Information Typing Architecture (DITA) Version 2.0

Technical Committee OASIS Darwin Information Typing Architecture (DITA) TC Chair Kristen James Eberlein (kris@eberleinconsulting.com), Eberlein Consulting LLC Editors Kristen James Eberlein (kris@eberleinconsulting.com), Eberlein Consulting LLC Robert D. Latest version: https://docs.oasis-open.org/dita/dita/v1.3/dita-v1.3-part3-all-inclusive.html Abstract The Darwin Information Typing Architecture (DITA) 2.0 specification defines both a) a set of document types for authoring and organizing topic-oriented information; and b) a set of mechanisms for combining, extending, and constraining document types.
