<xref>

A cross reference is an inline link. A cross reference can link to a different location within the current topic, another topic, a specific location in another topic, or an external resource such as a PDF or web page.

Attributes The following attributes are available on this element: link-relationship attributesLink relationship attributes include: format, href, type, and scope , universal attributesUniversal attributes include: audience, base, class, conaction, conkeyref, conref, conrefend, deliveryTarget, dir, id, importance, otherprops, outputclass, platform, product, props, rev, status, translate, and xml:lang , and keyref . Using keyref allows the link to be redirected to different resources when the topic is used in different contexts.
