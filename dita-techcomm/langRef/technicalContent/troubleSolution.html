<troubleSolution>

The <troubleSolution> element contains a possible cause and remedy for the problem described in a troubleshooting topic.

Usage information

The cause might be omitted if it is implicit or if the remedy is not associated with a cause. The remedy might be omitted if there is no known remedy for the cause.

Specialization hierarchy

The <troubleSolution> element is specialized from <bodydiv>. It is defined in the troubleshooting module.
