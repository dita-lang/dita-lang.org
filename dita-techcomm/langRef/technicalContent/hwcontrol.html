<hwcontrol>

The <hwcontrol> element represents the name of a key, button, switch, or other physical control on a device.

Usage information

Implementations that require different output for different types of hardware controls can use the @outputclass attribute to distinguish between different types of usage.

Specialization hierarchy

The <hwcontrol> element is specialized from <ph>. It is defined in the hardware domain module.

Attributes

The following attributes are available on this element: universal attributes and @keyref.

Example

The following code sample shows how <hwcontrol> can be used within task steps to identify hardware controls:

<step><cmd>If the command is already running, 
select the command window and press <hwcontrol>CTRL+C</hwcontrol>
to end processing.</cmd></step>
