<glossAbbreviation>

The <glossAbbreviation> element provides an abbreviated form of the term. An abbreviated form is typically formed by omitting some letters from a longer form.

Specialization hierarchy

The <glossAbbreviation> element is specialized from <title>. It is defined in the glossary entry module.

Attributes

The following attributes are available on this element: ID and conref attributes, localization attributes, @base, @class, @outputclass, and @rev.

Example

The following code sample shows how the <glossAbbreviation> is used in a glossary entry topic:

<glossentry id="adjective">
    <glossterm>adjective</glossterm>
    <glossdef>A word that describes a noun or noun phrase</glossdef>
  <glossBody>
    <glossAlt>
      <glossAbbreviation>adj</glossAbbreviation>
    </glossAlt>
  </glossBody>
</glossentry>
