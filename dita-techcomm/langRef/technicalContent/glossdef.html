<glossdef>

The <glossdef> element specifies the definition of one sense of a term.

Usage information

If a term has multiple senses, create a separate <glossentry> topic to define each sense.

Specialization hierarchy

The <glossdef> element is specialized from <abstract>. It is defined in the glossary entry module.

Attributes

The following attributes are available on this element: universal attributes.

Example

See the example in <glossentry>.
