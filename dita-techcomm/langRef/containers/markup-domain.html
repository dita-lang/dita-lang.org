Markup domain

The markup domain contains an element that can be used for the mention of named components in markup languages.

  • markupname
    The <markupname> element identifies named markup components, for example, elements or attributes in XML and SGML, named groups in XSD schemas, and named patterns in RELAX NG schemas.
