xNAL domain

The xNAL domain elements represent a subset of the Extensible Name and Address Standard. It is used to encode information about the author or authors of DITA information. The domain can be included in any DITA document type shell that requires additional metadata for names and addresses, although the implementations provided by OASIS only include it in the bookmap document type.

  • authorinformation
    The <authorinformation> element specifies detailed information about an author or authoring organization.
  • addressdetails
    The <addressdetails> element specifies information about the address of an author or authoring group.
  • administrativearea
    The <administrativearea> element specifies information about a county, state, or province.
  • contactnumber
    The <contactnumber> element specifies a contact number for an author or authoring organization, such as a telephone number.
  • contactnumbers
    The <contactnumbers> element provides a list of contact numbers.
  • country
    The <country> element specifies the name of a country.
  • emailaddress
    The <emailaddress> element specifies an e-mail address for an author or authoring organization.
  • emailaddresses
    The <emailaddresses> element provides one or more e-mail addresses for an author or authoring organization.
  • firstname
    The <firstname> element specifies the first name of an author.
  • generationidentifier
    The <generationidentifier> element specifies information about the generation of an author's name, for example Jr, III, or VIII.
  • honorific
    The <honorific> element specifies an author's title, such as Dr., Mr., or Ms..
  • lastname
    The <lastname> element specifies the last name of an author.
  • locality
    The <locality> element specifies information about a city, postal code, or ZIP code.
  • localityname
    The <localityname> element specifies the name of a locality or city.
  • middlename
    The <middlename> element specifies information about the middle name or middle initial of an author.
  • namedetails
    The <namedetails> element specifies detailed information about the name of an author or authoring organization.
  • organizationinfo
    The <organizationinfo> element specifies detailed information about an authoring organization.
  • organizationname
    The <organizationname> element specifies information about the name of an authoring organization.
  • organizationnamedetails
    The <organizationnamedetails> element specifies detailed information about the name of an authoring organization.
  • otherinfo
    The <otherinfo> element specifies additional detailed information about the name of an author or authoring organization.
  • personinfo
    The <personinfo> element specifies detailed information about an author, including name, address, and contact information.
  • personname
    The <personname> element specifies detailed information about the name of an author.
  • postalcode
    The <postalcode> element specifies information about a postal code or ZIP code for an author or authoring organization.
  • thoroughfare
    The <thoroughfare> element specifies for an author or authoring organization information about the thoroughfare, for example the street, avenue, or boulevard on which an address is located.
  • url
    The <url> element specifies a Uniform Resource Locator (URL) for an authoring organization.
  • urls
    The <urls> element specifies one or more Uniform Resource Locators (URLs) for an authoring organization.
