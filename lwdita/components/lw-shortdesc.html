Short description

A short description describes the purpose or main point of a topic.

Syntax

XDITA
<shortdesc>
HDITA
First element in article, if it is a paragraph, after title
MDITA (core and extended profiles)
First block, if it is a paragraph, after title

Usage information

When present in topics, the short description is the first paragraph of the topic. It can also be used for link previews and search results.

Rendering expectations

Processors SHOULD render the content of the short description component as the initial paragraph of the topic.

Attributes

The following attributes are available on this element: universal attributes.
