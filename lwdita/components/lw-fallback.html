Fallback

Fallback content is content to be presented when multimedia objects or other referenced content cannot be rendered.

Syntax

XDITA

<fallback>

HDITA

<???>

MDITA (core and extended profiles)

There is no specific support in MDITA core profile. If needed, use an HDITA snippet.

Processing expectations

The contents of this element are displayed only when the media that is referenced by the containing element cannot be displayed or viewed.

Attributes

The following attributes are available on this element: universal attributes.
