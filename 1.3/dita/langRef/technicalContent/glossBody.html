<glossBody>

The <glossBody> element is used to provide details about a glossary term (such as part of speech or additional forms of the term).

Content models

See appendix for information about this element in OASIS document type shells.

Inheritance

- topic/body concept/conbody glossentry/glossBody

Example

See the example in <glossentry>.

Attributes

The following attributes are available on this element: Universal attribute group (without the Metadata attribute group), @base from the Metadata attribute group, and outputclass.
