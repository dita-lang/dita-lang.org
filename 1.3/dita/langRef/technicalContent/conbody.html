<conbody>

The <conbody> element is the main body-level element for a concept.

Like the <body> element of a general <topic>, <conbody> allows paragraphs, lists, and other elements as well as sections and examples. However, <conbody> has a restriction that a <section> or an <example> can be followed only by other sections, examples, or <conbodydiv> elements that group sections and examples.

Attributes

The following attributes are available on this element: Universal attribute group (without the Metadata attribute group), @base from the Metadata attribute group, and outputclass.
