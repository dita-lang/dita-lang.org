<lcInteractionBase2>

Use <lcInteractionBase2> as the base for more specialized assessment types. This element replaces the <lcInteractionBase> element in DITA 1.2.

Content models

See appendix for information about this element in OASIS document type shells.

Inheritance

+ topic/div learningInteractionBase2-d/lcInteractionBase2

Example

Use lcInteractionBase2 only as a base for further specialization.

Attributes

The following attributes are available on this element: Universal attribute group and outputclass.
