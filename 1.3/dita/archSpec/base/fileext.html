File extensions

DITA uses certain file extensions for topics, maps, and conditional processing profiles.

Files that contain DITA content SHOULD use the following file extensions:

DITA topics
  • *.dita (preferred)
  • *.xml
DITA maps
*.ditamap
Conditional processing profiles
*.ditaval
