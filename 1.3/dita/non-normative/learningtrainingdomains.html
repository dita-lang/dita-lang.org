Learning domains and constraints: Where they are used

This section provides a summary of which document types use each of the learning and training domains and constraints.

Learning and training domains

Domain What includes it What does NOT include it
Learning (map/topic)
  • Learning and training document types: all topics
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all maps
Learning 2 (map/topic)
  • Learning and training document types: all topics
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all maps
Learning interaction base (map/topic)
  • Learning and training document types: all topics
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all maps
Learning interaction base 2 (map/topic)
  • Learning and training document types: all topics
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all maps
Learning metadata (map/topic)
  • Learning and training document types: all maps and topics
  • Base map, base topic
  • Technical content document types: all maps and topics
Learning map (map)
  • Learning and training document types: all maps
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all topics

Learning and training constraints

Constraint What includes it What does NOT include it
Learning aggregation <topicref> constraint
  • Learning and training document types: Learning group map, Learning object map
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all except Learning group map, Learning object map
No learning1 constraint
  • N/A (available for use with non-OASIS document types)
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all maps and topics
No learning2 constraint
  • N/A (available for use with non-OASIS document types)
  • Base map, base topic
  • Technical content document types: all maps and topics
  • Learning and training document types: all maps and topics
